# Python-SQLite3-Query-for-Hotel-Database
A Python SQL project that interacts with a SqLite3 hotel database, including information such as customer, employee, room and services. More information can be found in the Scenario Description.pdf file.

This project was written on Python version 3.9.10, SqLite library version 3.35.5 and DB Browser for SQLite version 3.13.1
